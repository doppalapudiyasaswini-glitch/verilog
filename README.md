This project demonstrates the design and verification of fundamental digital logic gates using Verilog HDL. Each gate is implemented as a separate module and verified using dedicated testbenches.
Logic                  Gate	Description
AND	             Output is high when all inputs are high
OR	             Output is high when at least one input is high
NOT              Inverts the input
NAND             Inverse of AND
NOR	             Inverse of OR
XOR	             Output is high when inputs are different
XNOR	           Output is high when inputs are same
Implemented basic logic gates using Verilog HDL with testbenches for functional verification
Simulated and validated designs using standard tools (ModelSim/EDA Play ground)
